Тестовое задание 
ADO.NET + ASP.NET + AngularJS



Тестовое задание

1. Создать страницу с реестром автомобилей, который содержит следующие поля:
- ФИО владельца
- Номер мобильного
- Номер автомобиля (Должна быть проверка на уникальность в базе).
- Марка автомобиля
- Модель автомобиля
Функционал – добавление/редактирование и удаление записи.
марка автомобиля и модель - зависимые списки (пример ZAZ и доступные модели
Sens, Lanos, и BMW и доступные модели 328,525,750)
2. Создать страницу с заполнением списка марок и моделей автомобилей.
Стек технологий: ASP.NET MVC 5 (Web API), Angular JS. Для обмена данными с БД
использовать ADO.NET и хранимые процедуры (stored procedure).
Нужно выполнить базовый функционал без заморочек с дизайном (веб часть).
